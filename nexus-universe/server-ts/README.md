# Projeto Aurora - Lord Eclipse

## Arquitetura Técnica
- **Linguagem Backend:** TypeScript (Node.js)
- **Engine Sugerida:** Godot
- **Comunicação:** WebSockets (Socket.io) + REST
- **Infraestrutura:** Kubernetes (GKE/EKS)
- **Banco de Dados:** Redis (Cache/Estado Rápido) & MongoDB (Persistência)

## Stack de Ferramentas (Instalada)
- **Monitoramento:** Prometheus & Grafana
- **Logging:** ELK Stack
- **CI/CD:** GitHub Actions, CircleCI, GitLab CI, Jenkins
- **Qualidade:** SonarQube
- **Infra:** Terraform & Ansible
- **Containerização:** Docker
